Problem / Risiko

Damit kritische Warnungen nicht unbemerkt bleiben.

Cyberangriffe werden oft nicht sofort bemerkt. Ein kompromittiertes Konto, ein infiziertes Gerät oder eine kritische Sicherheitswarnung kann unbemerkt bleiben, wenn niemand die Umgebung aktiv überwacht.

Viele Unternehmen verlassen sich darauf, dass Microsoft 365 schon sicher genug ist. Die Plattform bietet zwar starke Schutzfunktionen, diese müssen aber korrekt eingerichtet, gepflegt und regelmässig kontrolliert werden.

Einfach erklärt

Wichtige Begriffe ohne Fachsprache.

Security Baseline

Eine Security Baseline ist eine Sicherheitsgrundlage für Microsoft 365. Sie definiert, welche Schutzfunktionen mindestens aktiv sein sollten, damit Benutzer, Geräte, E-Mails und Daten besser geschützt sind.

Defender for Endpoint

Microsoft Defender for Endpoint ist ein moderner Schutz für Computer und Notebooks. Er erkennt nicht nur klassische Viren, sondern auch verdächtiges Verhalten, mögliche Angriffe und Sicherheitslücken auf Geräten.

Dark-Web-Monitoring

Im Dark Web werden unter anderem gestohlene Zugangsdaten gehandelt. Wenn E-Mail-Adressen oder Passwörter Ihrer Firma in Datenlecks auftauchen, kann dies ein Hinweis auf ein erhöhtes Risiko sein.

CVE

Eine CVE ist eine öffentlich bekannte Sicherheitslücke in Software oder Betriebssystemen. Sobald solche Lücken bekannt sind, können Angreifer gezielt danach suchen.
