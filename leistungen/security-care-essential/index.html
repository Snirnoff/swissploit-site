Problem / Risiko

Damit einfache Schwachstellen nicht zum Sicherheitsvorfall werden.

Viele Cyberangriffe beginnen nicht mit einer hochkomplexen Attacke, sondern mit einfachen Schwachstellen: einem gestohlenen Passwort, einem alten Benutzerkonto, einer falschen Freigabe oder einer Phishing-Mail.

Gerade kleine Unternehmen haben oft keine interne IT-Sicherheitsabteilung. Microsoft 365 wird täglich genutzt, aber die Sicherheit wird selten regelmässig überprüft. Genau hier setzt Security Care Essential an.

Einfach erklärt

Was bedeutet das konkret für Ihr Unternehmen?

MFA

MFA bedeutet Mehrfaktor-Authentifizierung. Ein Passwort allein reicht nicht mehr aus, um sich anzumelden. Zusätzlich braucht es zum Beispiel eine Bestätigung per App. Dadurch wird das Risiko stark reduziert, dass ein Angreifer mit einem gestohlenen Passwort Zugriff auf Ihre Firma erhält.

Externe Freigaben

Externe Freigaben entstehen, wenn Dateien, Ordner oder Teams-Inhalte mit Personen ausserhalb des Unternehmens geteilt werden. Solche Freigaben sind praktisch, können aber gefährlich werden, wenn sie nicht regelmässig kontrolliert werden.

Microsoft Secure Score

Der Microsoft Secure Score ist eine Sicherheitsbewertung von Microsoft. Er zeigt, wie gut Ihre Microsoft-365-Umgebung gemäss Microsoft-Empfehlungen geschützt ist. Wir bewerten diesen Wert verständlich und leiten daraus sinnvolle Empfehlungen ab.
